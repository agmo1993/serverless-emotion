The serverless emotion detection application that extracts facial features from an input image and outputs a corresponding emotion prediction.

Project Details
The emotion detection model here is an application of computer vision technology, which can extract facial features from an input image and output a corresponding prediction of the person's emotion in that image.

The initial model can detect a smile and seven different expressions: Angry, Disgust, Fearful, Happy, Neutral, Sad, and Surprised.

In this application, a simplified version is implemented with three basic emotions to be detecting: Happy, Neutral, and Sad. This model is developed on Python, using TensorFlow and OpenCV.
Our application is stateless and ephemeral. This makes it able to scale up and down as the load increased or decreases - and ideal for parallel execution.

The emotion detection model function is deployed using OpenFaas, an open-source framework for hosting serverless functions.

Every function is housed in a Docker container. It is managed using Docker Swarm, which can add more Docker containers when the function is called more often.
